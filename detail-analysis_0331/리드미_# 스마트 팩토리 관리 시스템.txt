# 스마트 팩토리 관리 시스템

ESP32-CAM과 AI 모델(YOLO, U-Net)을 활용한 실시간 불량 감지 시스템입니다. Flask 기반 웹 애플리케이션으로 생산 라인의 품질 관리를 혁신적으로 개선했습니다!

## 프로젝트 목적
- 생산 라인에서 배터리 불량을 실시간으로 감지하고 시각화.
- AI 기술로 불량 분석을 자동화하여 품질 관리 효율성 향상.
- 사용자 친화적인 웹 인터페이스로 관리자가 쉽게 모니터링 가능.

## 개발 단계

### 1단계: 실시간 영상 송출 및 배터리 탐지
- **작업**:
  - ESP32-CAM으로 실시간 영상 송출 구현.
  - YOLO 모델을 활용해 배터리 탐지 및 바운딩 박스 표시.
  - SocketIO로 실시간 영상 업데이트.
- **효과**:
  - 생산 라인에서 배터리를 실시간으로 감지.
  - YOLO 모델로 정확한 탐지 및 확률 표시 가능.

### 2단계: 불량 분석 및 시각화
- **작업**:
  - CNN 모델로 배터리 불량 여부 분류.
  - U-Net 모델로 불량 위치 시각화 및 점수 계산.
  - MySQL DB에 불량 데이터 저장.
- **효과**:
  - 불량 위치를 시각적으로 확인 가능.
  - 불량 점수로 심각도 판단, 품질 관리 정밀도 향상.

### 3단계: 웹 애플리케이션 통합
- **작업**:
  - Flask로 웹 애플리케이션 구축.
  - `dashboard.html`, `monitoring.html`, `detail-analysis.html`, `analysis.html` 구현.
  - 사용자 관리 및 로그인 시스템 추가.
- **효과**:
  - 관리자가 웹에서 실시간 모니터링 및 데이터 분석 가능.
  - 직관적인 UI로 사용 편의성 극대화.

## 프로젝트 장점
- **효율성**: 실시간 불량 감지로 품질 관리 시간 단축.
- **정확성**: YOLO와 U-Net 모델로 높은 탐지 및 분석 정확도.
- **확장성**: Flask와 MySQL 기반으로 기능 확장 용이.
- **사용성**: 사용자 친화적인 웹 인터페이스로 관리 편리.

## 기술 스택
- **백엔드**: Flask, SocketIO, MySQL
- **AI 모델**: YOLO, CNN, U-Net
- **프론트엔드**: HTML, CSS, JavaScript, Chart.js
- **하드웨어**: ESP32-CAM
